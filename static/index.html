? Formati riconosciuti e colonne estratte

Apache Combined / Nginx

IP, Ident, User, Timestamp, Method, Path, Protocol, Status, Size, Referer, UserAgent

Esempio: 192.168.1.1 - - [10/Oct/2023:13:55:36 +0000] "GET /api/status HTTP/1.1" 200 1234 "https://example.com" "curl/7.79"

Apache CLF (Common)

IP, Ident, User, Timestamp, Method, Path, Protocol, Status, Size

Esempio: 10.0.0.5 - alice [10/Oct/2023:14:01:00 +0000] "POST /login HTTP/1.1" 302 0

Syslog RFC 3164

Priority, Timestamp, Host, Process, PID, Message

Esempio: Oct 10 13:55:36 webserver sshd[12345]: Accepted publickey for alice from 192.168.1.100 port 54321

Syslog RFC 5424

Priority, Version, Timestamp, Host, AppName, ProcID, MsgID, StructuredData, Message

Esempio: <34>1 2023-10-10T13:55:36.003Z myhost app 12345 ID47 - Log message
